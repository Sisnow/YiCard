### 项目开发


###### **项目背景**

本项目旨在满足人们对记忆、背诵的自由度、舒适度的追求，实现记忆卡片的自定义、卡组的分类、统计信息以及时间提醒等功能。


###### **环境配置**

1. Android Studio Giraffe | 2022.3.1 Patch 1
2. minSdk 32
3. Java 11
